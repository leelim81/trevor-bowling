# 🎳👑 Trevor's Bowling

A bowling game designed by Trevor, where Trevor always wins.

Play it here once Pages is live: **https://leelim81.github.io/trevor-bowling/**

## How to play
- **Tap** (or click) for a straight throw — that's a **STRIKE!**
- **Swipe up** harder for more power; aim a little **left or right** to curve it.
- Stack up strikes to grow your **combo bonus** and climb the ranks:
  🥉 Bronze → 🥈 Silver → 🥇 Gold → 💎 Diamond.
- Trevor is **always #1** on the leaderboard. 👑

## Tech
- One self-contained `index.html` — no internet, no libraries, no build step.
- Vanilla JavaScript + HTML5 Canvas. Works on phone, tablet, and computer.
- Just open `index.html` in any browser, or host the folder anywhere static.
